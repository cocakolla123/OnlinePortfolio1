Maze Memory Game

AP Computer Science Applications

December 2020

The end of semester project for this class was to make an educational game. We wanted to make a game that would make memorizing facts more fun. Our idea was the Memory maze game. The Memory game was created with greenfoot. The point of the game is to answer all of the questions, which are stored in the cherries, while completing the maze. The maze would not finish unless you answer all of the questions correctly. Some challenges we faced while creating the game were that we did not know how to construct the maze in a fast and efficient way and we did not know how to make the person detect other objects. We worked around the construction problem by writing down the coordinates on a separate notes page, so we can save our progress while we were making the maze and add those coordinates to the initializing method. We figured out how to make the person detect other objects by searching the internet. We tweaked the code to make it work for our objects.
